# 🌦️ Weather App

A simple and responsive **Weather App** built using **HTML, CSS, and JavaScript**.  
This application allows users to check real-time weather information for any city with a clean and user-friendly interface.

The app fetches weather data using an API and displays temperature, weather conditions, and other details dynamically.

---

## 🌟 Features

- 🌍 Search weather by city name  
- 🌡️ Displays temperature and weather conditions  
- ☁️ Shows weather icons (cloudy, sunny, rain, etc.)  
- ⚡ Real-time data using API  
- 🎨 Clean and modern UI design  
- 📱 Fully responsive design (Mobile + Desktop)  
- 🚀 Fast and lightweight  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the app  
- **CSS3** – Styling and layout  
- **JavaScript** – API integration and functionality  
- **Weather API** – Fetch real-time data
